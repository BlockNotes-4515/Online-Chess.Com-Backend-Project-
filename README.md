## Getting Started
### Prerequisites
- Node.js and npm installed on your system.

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/real-time-multiplayer-chess.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd real-time-multiplayer-chess
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the server:**
   ```bash
   npm start
   ```
5. **Open the game in your browser:**
   Go to `http://localhost:3000` in two different tabs or windows to start playing.

## How to Play
1. **Player Roles**: When you open the game in a new tab, you'll be assigned either the white or black pieces. Your role will be displayed on the screen.
2. **Move Pieces**: Click and drag a piece to move it to your desired square. Legal moves will be validated automatically.
3. **Real-Time Updates**: Your moves will be reflected instantly on both players' screens.
